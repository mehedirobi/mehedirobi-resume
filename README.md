# 📄 Professional Resume (React + Vite)

A modern, responsive, and clean personal resume built using React and Vite.

This project is designed to present professional information in a structured, visually clean, and recruiter-friendly format.

---

## ✨ Features

- 📱 Fully responsive design (mobile, tablet, desktop)
- ⚡ Fast performance with Vite
- 🎨 Clean and modern UI layout
- 🧩 Component-based React architecture
- 🖨️ Print-friendly resume layout
- 🔗 Easy to update and maintain sections

---

## 🛠 Tech Stack

- React.js
- Vite
- JavaScript (ES6+)
- CSS / Tailwind CSS (if used)
- HTML5
